# CLAUDE.md

> Conciseness over grammar. Keep responses short.

## Commits

Format: `type(scope): message`

**Types**: `feat`, `fix`, `docs`, `refactor`, `test`, `chore`


**Rules**:
- Make multiple small commits if needed, not one big commit
- NEVER push commits - user pushes manually

**Examples**:
```
feat(api): add user endpoint
fix(web): resolve auth redirect
refactor(api): simplify handler logic
```
